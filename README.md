# Notepad-Tasks
An AI Agent generated app using Replit.

## Description
Manage daily tasks with features to add, delete, and check off completed items. A more realistic approach to tackle your tasks and start your day.

## Overview
This app is a notepad that is meant to be used as a To-Do List. Its purpose is to keep the user organized as they complete their tasks each day, and includes other features to reward the user for completing on time. I built this app to not only learn coding, but also to create a realistic tool that people use for time management and productivity.

## Features
Main Functionalities:
- Realistic Notepad with color options
- Write down tasks and checkmark them when completed
- Timer running in background
- Performance statistics
- Trophy Collection/Rewards based on efficiency
- Responsive layout (works on mobile and desktop)

## Tech Stack
- **Language:** Typescript
- **Frontend:** React, Vite, Tailwind CSS, Radix UI, React Hook Form
- **Backend:** Node.js, Express
- **Database:** Drizzle ORM
- **Tools:** drizzle-kit, npm, Git, Replit
